Real-World Vulnerability Examples

Capital One Data Breach (2019)

CVE-2019-11634: SSRF vulnerability allowed access to AWS metadata service, exposing temporary credentials and leading to massive data breach affecting 100+ million customers.

Parallel to Our Challenge:

Debug endpoint exposure → credential theft → internal system access

Equifax Breach (2017)

CVE-2017-5638: Apache Struts vulnerability combined with network segmentation failures allowed attackers to access sensitive databases containing 147 million records.

Parallel to Our Challenge:

Network segmentation bypass → database access → data exfiltration

Docker Hub Secret Exposure (2020)

Researchers found thousands of container images with embedded secrets, API keys, and database credentials exposed through environment variables and configuration files.

Parallel to Our Challenge:

Environment variable exposure → credential harvesting → system compromise
