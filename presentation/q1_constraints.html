Cloud Constraints and Decisions

Resource Constraints

Memory Limitations

Containerlab environments have limited memory allocation, requiring lightweight base images and optimized container configurations.

Network Complexity

Multiple network segments require careful IP addressing and routing configuration to avoid conflicts.

Design Solutions

Alpine Linux Base: Minimal footprint for router container
Static IP Assignment: Predictable addressing for educational clarity
Startup Scripts: Automated configuration for consistency

Performance Considerations

Technical Trade-offs

Container Count 4 containers (balanced complexity)
Network Segments 2 subnets (DMZ + Internal)
Service Exposure Single port (8080) for simplicity
Vulnerability Complexity Intermediate (realistic but solvable)
